# xd font data to css
xdのプラグイン。line-heightやletter-spacingを計算してコピーしてくれる。  
  
以下の記事を見て、自分で計算するのが面倒なので作った。  
https://note.com/lby/n/ne71f93c72702  
  
XDのプラグインのストアに提出してレビューしてもらっているところです。  

# 使い方
テキストにフォーカスした状態で以下のショートカットキーを叩く。  

## Cmd+Alt+A
フォントデータ全般をコピー

## Cmd+Alt+C
`letter-spacing`をコピー

## Cmd+Alt+L
`line-height`をコピー
